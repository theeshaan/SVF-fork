This folder contains some good testcases to test pointer analysis output after I make changes

To test on a particular testcase, copy the code into `../my_test.cpp` and in the `my_stuff` directory, run:

```bash
./run_extractor.sh --svf-path ../Release-build/bin --llvm-path ../llvm-18.1.0.obj/bin/ my_test.cpp > test_output.txt
```

The points-to output generated by SVF will be in `my_stuff/my_test.pta`, and the extracted points-to output will be in `my_stuff/test_output.txt`.
